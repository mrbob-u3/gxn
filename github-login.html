To make this real

This is a working UI demo. To wire up actual GitHub login, you need to:

1. Go to github.com/settings/developers
2. Create a new OAuth App
3. Set your callback URL
4. Grab your Client ID & Client Secret
5. Add them to your backend (Node, Python, etc.)
